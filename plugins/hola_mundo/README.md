# Plugin Hola Mundo

Plugin de demostración para el portal público de FSFramework.

## Características

- Muestra "Hola Mundo" en el portal público
- No requiere base de datos
- Ejemplo de cómo los plugins pueden registrar contenido público

## Instalación

1. Activa el plugin desde el panel de administración
2. Visita la página de inicio (portal) para ver el contenido

## Funcionamiento

Este plugin registra una sección pública mediante el archivo `portal_section.php`, 
que es detectado automáticamente por el plugin `portal`.
