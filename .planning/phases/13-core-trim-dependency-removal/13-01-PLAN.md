# Plan 01: Core Trim & Dependency Removal

Remove API-only packages from root `composer.json` after Phase 12 plugin vendor is in place. Audit `src/Api/` and the rest of `src/` for orphaned API runtime code. Keep `api.php` as thin bootstrap.

**Prerequisite:** Phase 12 complete — `plugins/api_base/vendor/` provides `OpenApi\Generator`.

## must_haves

- Root `composer.json` no longer lists `zircote/swagger-php` or `firebase/php-jwt`
- Root `composer.lock` regenerated via `ddev exec composer update` (or equivalent) without those packages
- `src/Api/` contains only `Attribute/`, `Auth/Contract/`, and `Exception/` — no runtime/router/middleware classes
- No API runtime classes remain elsewhere under `src/` (e.g. no `ChainedAuthAdapter`, `ApiRuntime`, middleware under core)
- `api.php` unchanged in behavior: 404 when `api.runtime` missing; delegates to plugin when active
- With `api_base` active + plugin vendor installed, `OpenApi\Generator` resolves via plugin autoload (not root vendor)
- `ddev exec php vendor/bin/phpunit` passes (full core suite)
- `ddev exec php vendor/bin/phpunit -c plugins/api_base/phpunit.xml` passes

## Verification

```bash
ddev exec composer update zircote/swagger-php firebase/php-jwt --with-all-dependencies 2>/dev/null || ddev exec composer update --no-interaction
grep -E 'swagger-php|php-jwt' composer.json && exit 1 || true
ddev exec php -r "
require 'vendor/autoload.php';
require 'plugins/api_base/vendor/autoload.php';
exit(class_exists('OpenApi\\\\Generator') ? 0 : 1);
"
ddev exec php vendor/bin/phpunit
ddev exec php vendor/bin/phpunit -c plugins/api_base/phpunit.xml
```

## Out of scope (Phase 14)

- Root `AGENTS.md` / `README.md` dependency table updates (DOC-01)
- `.planning/codebase/STACK.md` updates (DOC-02)
- `tests/Api/` suite cleanup (TEST-01/02)

## <threat_model>
- **Threats:** Low — removing unused root dependencies. Risk: something in core still referenced swagger-php or JWT from root vendor only.
- **Mitigation:** grep workspace for `OpenApi\`, `Firebase\JWT`, `zircote` before/after; run full PHPUnit.
- **ASVS L1:** V8.1.1 — minimize attack surface by dropping unused packages from core lockfile.
</threat_model>
